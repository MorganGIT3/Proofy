# Proofy Extension Popup Design

Design de popup magnifique pour l'extension Proofy, inspiré du style de la landing page.

## Structure

- **Popup.tsx** - Composant principal qui gère la navigation entre les pages
- **WelcomePage.tsx** - Première page avec logo, texte et bouton "Fake it"
- **LoginPage.tsx** - Page de connexion avec Google, Apple et Email
- **RedirectPage.tsx** - Page de redirection après connexion
- **SettingsPage.tsx** - Page de paramètres avec des boutons à cocher stylisés

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Caractéristiques

- Design moderne avec coins arrondis
- Animations fluides avec Framer Motion
- Style cohérent avec la landing page Proofy
- Effets de glow orange dans le fond
- Boutons avec effets shimmer et hover
- Transitions entre les pages

## Pages

1. **Welcome Page** : Logo Proofy, texte principal et bouton "Fake it"
2. **Login Page** : Options de connexion (Google, Apple, Email)
3. **Redirect Page** : Page de redirection avec message de succès et barre de progression
4. **Settings Page** : Paramètres avec checkboxes stylisées (Editor Mode, Auto-save, etc.)
